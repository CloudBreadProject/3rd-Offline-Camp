4.    Facebook SDK 를 이용한 로그인 구현

 

①      Facebook에 앱 추가하기

②      Azure Mobile App 에 페이스북 앱 추가하기

I.          Facebook 개발자 웹 사이트로 이동한 다음 Facebook 계정 자격 증명으로 로그인합니다.

II.        (선택 사항) 아직 등록하지 않은 경우 My Apps, Register as a Developer를 차례로 클릭하고 정책에 동의한 후 등록 단계를 따릅니다.

III.       My Apps > Add a New App > Website > Skip and Create App ID를 클릭합니다.

IV.       Display Name에 앱의 고유한 이름을 입력하고 앱에 대한 Category를 선택한 다음 Create App ID를 클릭하고 보안 검사를 완료합니다. 새 Facebook 앱에 대한 개발자 대시보드로 이동됩니다.

V.        App Secret 필드에서 Show를 클릭하고 요청될 경우 암호를 제공한 다음 App ID 및 App Secret 값을 적어 둡니다. 나중에 이 값을 사용하여 Azure에서 응용 프로그램을 구성합니다._- AZURE.NOTE - 참고:_보안 정보 앱 암호는 중요한 보안 자격 증명입니다. 다른 사람과 이 암호를 공유하거나 클라이언트 응용 프로그램 내에 배포하지 마세요._

 

왼쪽 탐색 모음에서 Settings를 클릭하고, App Domains에 모바일 서비스의 도메인을 입력하고, 선택적 Contact Email을 입력하고, Add Platform을 클릭하고, Website를 선택합니다.

 

VI.       왼쪽 탐색 모음에서 Settings를 클릭하고, App Domains에 모바일 서비스의 도메인을 입력하고, 선택적 Contact Email을 입력하고, Add Platform을 클릭하고, Website를 선택합니다.

 

VII.     Site URL에 모바일 서비스의 URL을 입력하고 Save Changes를 클릭합니다.

VIII.    Show를 클릭하고 요청될 경우 암호를 제공한 후 App ID 및 App Secret 값을 기록해 둡니다.

 

 

IX.       Advanced 탭을 클릭하고, Valid OAuth redirect URIs에 아래URL 형식 입력한 다음 Save Changes를 클릭합니다.

https://<mobile_service>.azure-mobile.net/login/facebook

 

아래 사이트에서 위 내용에 대해 자세하게 볼 수 있습니다.

https://azure.microsoft.com/ko-kr/documentation/articles/mobile-services-how-to-register-facebook-authentication/
